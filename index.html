Actividad 1 (Valor 10 puntos) Investiga en diferentes sitios web y describe cada tema:

1. Definision de CSS

CSS es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML. Permite a los desarrolladores controlar el estilo y el diseño de múltiples páginas web a la vez.

2. Funcionamiento Básico de CSS.

CSS funciona aplicando estilos a elementos HTML. Utiliza reglas que consisten en selectores y declaraciones para determinar cómo se verá cada elemento.

3. Incluir CSS en un archivo HTML.

Hay tres formas de incluir CSS en HTML:

  • Inline CSS: Se utiliza el atributo style directamente en el elemento HTML.
  • Internal CSS: Se coloca dentro de un bloque (style) en la sección (head).
  • External CSS: Se enlaza un archivo .css externo utilizando la etiqueta (link).

4. Definición de una regla CSS.

Una regla CSS es un conjunto de estilos aplicados a uno o más elementos HTML. Se compone de un selector y un bloque de declaración.

5. Elemento de una regla CSS.

  • Regla: La combinación del selector y el bloque de declaración.
  • Selector: Indica qué elementos HTML se verán afectados.
  • Bloque de declaración: Contiene una o más declaraciones de estilo.
  • Propiedad: La característica del estilo que se desea cambiar (e.g., color, font-size).
  • Valor: El valor que se asigna a la propiedad (e.g., red, 16px).

6. Medios CSS.

Los medios en CSS permiten aplicar estilos específicos según el tipo de dispositivo o el tamaño de la pantalla. Se utilizan media queries para definir estos estilos.

7. Comentarios CSS.

Los comentarios en CSS se utilizan para anotar el código y no se muestran en el navegador. Se escriben entre /* y */.

8. Sintaxis de la definición de las propiedades CSS.

La sintaxis de una propiedad CSS incluye el nombre de la propiedad seguido de dos puntos (:), el valor y un punto y coma (;).
